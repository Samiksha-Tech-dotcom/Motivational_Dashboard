# 🌌 Motivational Dashboard (React)

A modern, aesthetic **Motivational Quotes Dashboard** built using React.  
It fetches quotes from an API and provides an interactive UI with glowing effects, likes system, and a beautiful **starry night theme** ✨

---

## 🚀 Features

- 🌐 Fetch random quotes from API
- ❤️ Like / Unlike quotes
- 👎 Dislike (skip) quotes
- 📋 Copy quote to clipboard (with "Copied!" feedback)
- 💾 Persistent liked quotes using localStorage
- ❌ Remove liked quotes
- 📊 Total liked quotes count
- ✨ Sparkling starry night background with twinkling effect
- 🎨 Modern glassmorphism UI with glowing elements

---

## 🛠️ Tech Stack

- React (useState, useEffect)
- CSS (Glassmorphism + Animations)
- LocalStorage
- Fetch API

---
